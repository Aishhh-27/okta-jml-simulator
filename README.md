# Okta JML Simulator

A Python-based Identity Lifecycle Management (JML) simulator that automates employee identity lifecycle operations in Okta based on HR-style events.

The project demonstrates how Joiner, Mover, and Leaver events can be processed automatically using Python and the Okta REST API.

---

## Project Overview

In a real organization, employee lifecycle events are usually generated by an HR system.

For example:

- A new employee joins the company
- An employee changes departments or roles
- An employee leaves the company

These events need to trigger corresponding identity actions.

This project simulates that process:

```text
HR Lifecycle Event
        |
        v
Python Event Processor
        |
        v
OAuth 2.0 + JWT + DPoP Authentication
        |
        v
Okta REST API
        |
        +------------------+
        |                  |
        v                  v
     JOINER             MOVER
   Create User       Update Profile
        |
        v
     LEAVER
   Suspend User
